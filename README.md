# 📚 Biblioteca Gerenciável

Sistema para gerenciamento de biblioteca, permitindo o controle de livros, usuários e empréstimos de forma simples e eficiente.

# 📌 Sobre o Projeto

O Biblioteca Gerenciável é uma aplicação desenvolvida com o objetivo de facilitar a organização de acervos bibliográficos, permitindo:

Cadastro e gerenciamento de livros
Controle de usuários
Registro de empréstimos e devoluções
Consulta rápida de informações

Esse tipo de sistema é amplamente utilizado para automatizar processos em bibliotecas e melhorar o controle de acervo .

🚀 Funcionalidades
📖 Cadastro de livros
🔍 Busca de livros
👤 Cadastro de usuários
🔐 Controle de acesso (se aplicável)
📦 Empréstimo de livros
🔄 Devolução e renovação
📊 Listagem de acervo
